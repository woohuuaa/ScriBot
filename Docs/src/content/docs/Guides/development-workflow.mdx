## Getting Started

**Use Docker Compose for development:**

```bash
cd KDAI-DockerCompose
cp .env.example .env
docker compose up -d
```

**Access points:**

- Application (Frontend + API): http://localhost:8001
- MinIO Console: http://localhost:9001

## Essential Commands

### Backend (Laravel)

```bash
# In container
docker compose exec backend bash

# Common operations
php artisan migrate
php artisan db:seed
php artisan queue:work
php artisan tinker

# Clear caches
php artisan cache:clear
php artisan config:clear
php artisan route:clear
```

### Frontend (Vue)

```bash
# In container
docker compose exec frontend bash

# Install dependencies (if needed)
npm install

# Development server runs automatically
```

### Services (Python)

```bash
# Check service logs
docker compose logs -f qes
docker compose logs -f atts-csp
docker compose logs -f atts-ts

# Restart service
docker compose restart qes
```

### Database

```bash
# Access PostgreSQL
docker compose exec postgres psql -U kdai_user -d kdai

# Reset database
docker compose exec backend php artisan migrate:fresh --seed
```

## Debugging

### View Logs

```bash
# All services
docker compose logs -f

# Specific service
docker compose logs -f backend
docker compose logs -f frontend
```

### Check Service Health

```bash
# All containers
docker compose ps

# Test API
curl http://localhost:8001/api/health
```

### Database Issues

```bash
# Check database connection
docker compose exec backend php artisan tinker
>>> DB::connection()->getPdo()

# View recent queries
docker compose exec backend php artisan tinker
>>> DB::enableQueryLog(); // run some code; dd(DB::getQueryLog());
```

## Code Changes

### Backend Changes

```bash
# Restart PHP-FPM
docker compose restart backend

# Or reload with new code
docker compose exec backend php artisan config:cache
```

### Frontend Changes

```bash
# Hot reload works automatically
# If issues, restart container
docker compose restart frontend
```

### Service Changes

```bash
# Rebuild and restart service
docker compose up -d --build qes
```

## Testing

### Frontend Tests


## Common Issues

### Services Not Starting

```bash
# Check for port conflicts
docker compose ps
lsof -i :8000

# View detailed logs
docker compose logs backend
```

### Database Connection Failed

```bash
# Check PostgreSQL is running
docker compose ps postgres

# Reset database
docker compose exec backend php artisan migrate:fresh
```

### Frontend Not Loading

```bash
# Check if built correctly
docker compose exec frontend npm run build

# Clear browser cache
# Check network tab for API calls
```

## Development Tips

- **Use the Docker environment** - it's configured correctly
- **Check logs first** when something doesn't work
- **Restart services** after config changes
- **Use `docker compose exec`** to run commands in containers
- **Database changes require migration** - don't modify schema directly

## Git Workflow

### Branch Strategy

**Feature Branches:**

```bash
git checkout -b feature/add-debate-questions
# Make changes
git commit -m "Add debate questions functionality"
git push origin feature/add-debate-questions
```

**Pull Request Process:**

1. Create PR with description
2. Run tests locally
3. CI/CD checks pass
4. Code review
5. Merge to main

### Commit Conventions

**Format:**

```
type(scope): description

[optional body]

[optional footer]
```

**Types:**

- `feat`: New feature
- `fix`: Bug fix
- `docs`: Documentation
- `style`: Code style
- `refactor`: Code refactoring
- `test`: Testing
- `chore`: Maintenance

**Examples:**

```
feat(debates): add question extraction endpoint
fix(auth): resolve token refresh issue
docs(api): update endpoint documentation
```

## Troubleshooting Common Issues

### Backend Issues

**Class Not Found:**

```bash
composer dump-autoload
```

**Permission Issues:**

```bash
chmod -R 775 storage/
chmod -R 775 bootstrap/cache/
```

**Database Connection:**

```bash
# Check PostgreSQL
psql -h localhost -U your_user -d kdai_dev

# Test Laravel connection
php artisan tinker
>>> DB::connection()->getPdo()
```

### Frontend Issues

**Port Already in Use:**

```bash
lsof -ti:5173 | xargs kill -9
```

**CORS Issues:**

- Check `config/cors.php`
- Verify `APP_URL` in `.env`

### Service Issues

**Port Conflicts:**

- Check service ports in docker-compose.yml
- Ensure no other services running on ports

**Dependency Issues:**

```bash
# Python
pip install --upgrade pip
pip install -r requirements.txt --force-reinstall
```

**Redis Connection:**

```bash
redis-cli ping
# Should return PONG
```
