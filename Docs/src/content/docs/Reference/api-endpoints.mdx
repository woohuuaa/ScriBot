## Authentication Overview

All API endpoints require authentication except health checks.

### Authentication Methods

- **Sanctum Tokens**: For user authentication
- **API Keys**: For service-to-service communication

### Authentication Flow

<MermaidChart
  chartDefinition={`sequenceDiagram

    participant Sanctum

    Note over Client,DB: User Login Flow
    Client->>API: POST /api/auth/login
    API->>DB: Validate credentials
    DB-->>API: User data
    API->>Sanctum: Generate token
    Sanctum-->>API: Sanctum token
    API-->>Client: Token + user data

    Note over Client,DB: Authenticated API Calls
    Client->>API: Request with Bearer token
    API->>Sanctum: Validate token
    Sanctum-->>API: Token valid
    API->>DB: Process request
    DB-->>API: Response data
    API-->>Client: JSON response

    Note over Client,DB: Service Communication
    Client->>API: Request with API Key
    API->>DB: Process service request
    DB-->>API: Response data
    API-->>Client: JSON response

`} />

### Headers

```http
Authorization: Bearer {sanctum_token}
X-API-Key: {api_key}
Content-Type: application/json
```

## Public Endpoints

### Health Check

```http

```

**Response:**

```json
{

```

## Authentication Endpoints

### Login

```http
POST /api/auth/login
```

**Request:**

```json
{
  "email": "user@example.com",
  "password": "password"
}
```

**Response:**

```json
{
  "token": "sanctum_token",
  "user": {
    "id": "uuid",
    "name": "User Name",
    "email": "user@example.com"
  }
}
```

### Logout

```http
POST /api/auth/logout
Authorization: Bearer {token}
```

### Refresh Token

```http
POST /api/auth/refresh-token
Authorization: Bearer {token}
```

## Error Responses

All endpoints return consistent error format:

```json
{
  "message": "Error description",
  "errors": {
    "field": ["Validation error details"]
  }
}
```

### HTTP Status Codes

- `200` - Success
- `201` - Created
- `400` - Bad Request
- `401` - Unauthorized
- `403` - Forbidden
- `404` - Not Found
- `422` - Validation Error
- `500` - Internal Server Error

## Rate Limiting

API endpoints may be rate limited:

- **User Authentication**: 10 requests per minute
- **Service Endpoints**: 100 requests per minute
- **Data Operations**: 60 requests per minute

Rate limit headers are included in responses:

```http
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 59
X-RateLimit-Reset: 1642694400
```
